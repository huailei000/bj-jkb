## 使用说明

- 可以在目录主页面整体作为 PWA (Progressive Web App) 安装，以达到全屏显示的效果
  - 在 Safari (iOS) 上，点击下方“分享”按钮，再点击“添加至主屏幕”
  - 在 Chrome (Android) 上，点击右上角菜单按钮，再点击“添加至主屏幕”
  - 在 Firefox (Android) 上，点击右上角菜单按钮，再点击“安装”
- 亦可以在各项目页面单独安装某一项目
  - 如果遇到状态栏文字颜色问题，请尝试在常用项目子页面中单独安装
